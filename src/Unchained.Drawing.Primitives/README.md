# Unchained.Drawing.Primitives

MIT-licensed, pure-managed primitive types for the Unchained document-processing suite — color and geometry value types shared across the drawing, PDF, and OOXML layers. No native dependencies, no managed dependencies.

**Targets:** `net8.0` · `net9.0` · `net10.0`
**License:** MIT

---

## You usually do not need to install this package directly

`Unchained.Drawing.Primitives` is an automatic transitive dependency of the higher-level Unchained packages (`Unchained.Drawing`, `Unchained.Ooxml`, and the rendering packages). Install one of those instead.

---

## What's inside

Lightweight, allocation-friendly value types for colors and geometry (points, sizes, rectangles) used as the common vocabulary between the suite's drawing and document layers.
